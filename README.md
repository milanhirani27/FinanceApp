## 1. Clone Repository
```bash

## 2: Install Dependencies

```bash
# using npm
npm install

# OR using Yarn

```


### 3: Start Metro

```bash
# using npm
npm start

# OR using Yarn
yarn start
```


## 4: Start your Application

### For Android

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# install pods
cd ios
pod install
cd ..

# using npm
npm run ios

# OR using Yarn
yarn ios
```
